28  // RSD removal for particles in a periodic box
29  // We use a fixed line of sight direction los_direction, typically
30  // coordinate axes (e.g. los_direction = {0,0,1} )
31  // We are solving the equation LPT equation
32  // D(bPsi) + beta D((bPsi*r)r) = -delta_observed
33  // and assuming zero-curl (which is not perfectly true) of the second term
34  // so we can use Fourier methods
35  // Then we subtract the RSD which is given by Psi_rsd = beta (bPsi*r)r
36  // We then iterate this procedure niterations times
37  // Here beta = f / b growth-rate over galaxy bias
38  // smoothing_options = {smoother_filter, smothing_scale} with smothing_scale is units of the box
39  // The smoothing filter is an (optional) filter we apply to the density field in
40  // k-space. Options here are sharpk, tophat, gaussian. If R < gridspacing then no
41  // smoothing will be done
42  //
43  // This also works for a survey. Then los_direction is the observer position
44  // and we need to have a box big enough so that particles don't wrap around
45  // when shifted!
46  //
47  // This method assumes scale-independent growth factor, but that is easily changed
48  //
49  //============================================================================
