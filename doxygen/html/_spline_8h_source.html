18 //====================================================
19 //
20 // This is a wrapper class for easy use of GSL splines
21 // It is (OpenMP) thread safe as long as splines are
22 // not created by a subthread and later tried to be used
23 // by more than that thread!
24 //
25 // The fiducial splines are cubic splines for 1D and
26 // bicubic for 2D assuming natural (y''=0) boundary
27 // conditions at the end-points
28 //
29 // The fiducial choice allows to get the function value
30 // and up to second order derivatives. NB: if you change
31 // the fiducial choice to a lower order spline,
32 // e.g to gsl_interp_linear, then the second derivative
33 // deriv_xx etc. will not work
34 //
35 // If evaluating out of bounds it will return the closest
36 // value. To get a warning when this happens call
37 // set_out_of_bounds_warning(true) before using the spline
38 // or set the define SPLINE_FIDUCIAL_SPLINE_WARNING to be true
39 //
40 // Errors are handled via the throw_error function
41 //
42 // Compile time defines:
43 // USE_MPI : Use MPI (only effect is in on how errors are handled)
44 // USE_OMP : Use OpenMP
45 // SPLINE_FIDUCIAL_INTERPOL_TYPE : Cubic spline is the fiducial choice
46 // SPLINE_FIDUCIAL_INTERPOL_TYPE_2D : Bicubic is the fiducial choice
47 // SPLINE_FIDUCIAL_SPLINE_WARNING : Show warnings if we evaluate out of bounds
48 //
49 //====================================================
